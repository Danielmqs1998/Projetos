## **W-ERP**
Sistema de ERP básico com funcionalidades como:

- Autenticação de usuário com token temporário
- Cadastro e manutenção de clientes e produtos (CRUD)
- Listagem sob demanda
- Pré-visualização de imagens
- Filtros em tempo real

## **W-STORE**
Plataforma de e-commerce simples com funcionalidades como:

- Autenticação de usuário
- Integração de produtos com o ERP
- Listagem de produtos
